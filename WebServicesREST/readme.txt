# trustedcompany
This project contains repo of example REST webservice developed for TrustedCompany Assignment

1. Its a simple REST Webservice developed using Jersey.
2. It returns a simple static JSON string,
3. The redirection happens using annotations.

Test URL's - from locally deployed on Tomcat
http://localhost:8080/WebServicesREST/ws/TrustedCompany/example
http://localhost:8080/WebServicesREST/
